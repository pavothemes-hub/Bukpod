# Elementor Widgets

Once **Elementor** and **Pavo Core** are active, Bukpod adds its own set of widgets to the Elementor editor panel (search for "Bukpod" or look under the **Bukpod** category). Use this page as a quick reference for what each one does.

| Widget | What it's for |
|---|---|
| Bukpod Site Logo | Displays your site logo/favicon from Customizer > Site Identity. |
| Bukpod Nav Menu | Renders any WordPress menu — used for the main and footer navigation. |
| Bukpod Menu Canvas | The off-canvas mobile menu (trigger icon, panel position, content). |
| Bukpod Breadcrumbs | Auto-generated breadcrumb trail with an optional page title. See [Breadcrumb](../setting/breadcrumb.md). |
| Posts Grid | Grid/list of blog posts with filters. |
| Bukpod Archive Post | Blog archive listing layout (used inside Archive templates). |
| Icon Box (Bukpod Icon Box) | Icon + heading + description feature block. |
| Bukpod Steps | Numbered process/step-by-step block. |
| Bukpod Pricing | Pricing table with feature list and CTA button. |
| Bukpod Testimonials | Testimonial slider/grid. |
| Bukpod Brands | Logo/partner carousel. |
| Bukpod Slides | Nested slider / carousel container. |
| Bukpod Toggle | Accordion / toggle content block. |
| Bukpod Slider Scrolling | Continuous scrolling ("marquee") text or logo strip. |
| Bukpod Scroll Image | Image with a scroll-driven animation effect. |
| Image Hotspots | Clickable hotspots over an image (supports linking to WooCommerce products). |
| Bukpod Video Popup | Button/thumbnail that opens a video in a lightbox popup. |
| Bukpod Quick Action | Floating quick-action buttons (e.g. call, chat, book now). |
| Language Switcher | WPML language switcher, styled to match the theme. |
| Bukpod Contact Form | Elementor-styled wrapper around a Contact Form 7 form. See [Set up Contact Form](set-up-contact-form.md). |
| MailChimp Sign-Up Form | Elementor-styled wrapper around an MC4WP sign-up form. See [Set up MailChimp](set-up-mailchimp.md). |
| Bukpod Service | Grid/list of Service posts. See [Service](service/README.md). |
| Bukpod Service Meta | Meta details (price, duration, etc.) on a single Service page. |
| Bukpod Archive Services | Service archive listing layout (used inside Archive templates). |
| Bukpod Team Box | Team member grid built from a repeater — no separate post type. See [Teams](teams/README.md). |
| Bukpod Products | WooCommerce product showcase. |
| Bukpod Category Product | WooCommerce product-category showcase. |

{% hint style="info" %}
Bukpod also extends a few native Elementor widgets — **Accordion**, **Button**, **Image Box**, and **Tabs** — with extra style controls, so you'll see additional design options on those without needing a separate widget.
{% endhint %}
