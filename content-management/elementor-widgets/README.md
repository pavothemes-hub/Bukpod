## Native widget extensions

Bukpod also extends 4 of Elementor's own native widgets with extra controls, so you'll see additional options on them without needing a separate Bukpod widget:

* **Accordion** (Nested Accordion) - adds icon Width/Height, Border Radius, Border (+ Hover/Active border color), and Background Color (Normal/Hover/Active) to the title icon's Style tab.
* **Button** - adds a **Type** select on the Content tab (`Default` / `OutLine` / `Info` / `Success` / `Warning` / `Danger` / `Link`) plus dedicated icon color controls. See [Button](../../setting/button.md) for how this interacts with the global button style.
* **Image Box** - adds a Box Shadow control on the image itself, and fixes the title's hover color control to also apply on keyboard focus.
* **Tabs** (Nested Tabs) - adds a Margin control for the tab-heading row.

{% hint style="info" %}
These are style-only additions on top of Elementor's stock widgets - there's nothing new to configure on the Content tab except the Button's **Type** select.
{% endhint %}
