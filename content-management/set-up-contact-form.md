# Set up Contact Form

**PavoThemes** uses plugin [Contact Form 7](https://wordpress.org/plugins/contact-form-7/) to help you create a contact form - It's free.

**Step 1 - I**nstall **Contact Form** for WordPress.

<figure><img src="../.gitbook/assets/image (288).png" alt=""><figcaption></figcaption></figure>

**Step 2 -** After installing this plugin, go to **Contact > Contact Forms > Add New/ Edit a Contact Form** and use the following this mark-up and paste it to your form (Of course, you can edit the text "Name", "Email", "Subject", "Your Message", "Submit" to the text as you want):
