# Set up Header

By bringing the power of the Elementor editor to the header/footer areas, you can design them any way you like, full-width included, and be able to complete projects faster than ever. You can also save header and footer templates, and reuse them on any other project.

If you are beating yourself up right now for all the hours you lost in the past over your header design, we share your feelings. From now on, with the help of Theme Builder, things are going to be much faster.

**Step 1 -** Go to **WordPress Dashboard > Theme Builder > Add New**, name the template, set **Type of Template** to **Header**, then click **Edit with Elementor**.

**Step 2 - Logo -** Add the **Bukpod Site Logo** widget to pull in the logo set under Appearance > Customize > Site Identity.

**Step 3 -** Menu **-** Add the **Bukpod Nav Menu** widget and select which WordPress menu it should display (see [Set up Menu](set-up-menu.md)).

**Step 4 -** Edit Button **-** Drop in a **Bukpod Button** widget for a header call-to-action, if your design needs one.

**Step 5 -** Menu Canvas Mobile **-** Add the **Bukpod Menu Canvas** widget to control the off-canvas mobile menu (trigger icon, position, and content).

**Step 6 -** Text Editor **-** Use a standard Elementor Text Editor widget for any extra header text, such as a phone number or tagline.

## How are Conditions Displayed?

Back in **Theme Builder**, edit the template's **Display On** and **User Roles** settings to choose where it should appear — for example **Entire Website**, or specific pages/post types only.

Click **Publish**, then **Save & Close**.
