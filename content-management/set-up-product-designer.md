# Set up Product Designer

Bukpod bundles **WooCommerce Designer Pro** (WCDP) - the tool that lets customers upload their own artwork, add text/cliparts, and preview it on your product (t-shirts, mugs, tote bags, etc.) before adding to cart. This is the core of the print-on-demand shopping experience.

{% hint style="info" %}
Make sure **WooCommerce Designer Pro** is installed and active (**Appearance > Theme Setup** if it wasn't installed yet). Its own settings menu appears in the Dashboard sidebar as **WC Designer Pro**.
{% endhint %}

## 1. Create a Design

**Step 1 -** Go to **WC Designer Pro > Designs > Add new design**.

**Step 2 -** Under **Select parameter**, choose (or create, under **WC Designer Pro > Parameters**) the canvas size/print area that matches your product (e.g. front print area of a t-shirt).

**Step 3 -** Use the **Design** editor box to build the base template - upload your product mockup image, position printable zones, and add any default text/clipart placeholders customers will see when they open the designer.

**Step 4 -** Organize designs with **WC Designer Pro > Categories** if you offer many templates (e.g. one category per product type).

## 2. Add supporting assets (optional)

Under the **WC Designer Pro** menu you can also manage what customers can pick from inside the design editor:

* **Fonts** - upload custom fonts for the text tool.
* **Shapes** and **Cliparts** - preset shapes/graphics customers can drag onto their design.
* **Image filters** - preset photo filters/effects available in the editor.

## 3. Enable the Designer on a product

**Step 1 -** Edit any WooCommerce product, then find the **WCDP - Custom Product** box in the sidebar.

**Step 2 -** Check **Enable Custom Product**.

**Step 3 -** Under **Select main design**, choose the Design you created in Step 1. You can also attach multiple designs (individually, or by category) so the shopper can switch templates inside the editor.

**Step 4 -** Click **Update/Publish** on the product. A **Personalize** button (or the editor itself) now appears on that product's page.

{% hint style="info" %}
For **variable products** (e.g. products with a Color attribute), open the product's **Attributes > WCDP - Actions** tab to set a different front/back image, background color, canvas size, and print/PDF output size per variation - useful when each color needs its own print file dimensions from your production partner.
{% endhint %}

## 4. Customize the editor (optional)

**Step 1 -** Go to **WC Designer Pro > Settings**.

**Step 2 -** Use the **General** tab for editor behavior, **Shortcut Keys** for keyboard shortcuts, **Style** to reskin the editor's colors/layout to match your brand, and **License** to activate your copy of the plugin with the license key from your purchase.

{% hint style="info" %}
Orders placed with a custom design attach the customer's saved design file to the order - check **WC Designer Pro > Designs** or the order screen to download the print-ready file for production/fulfillment.
{% endhint %}
