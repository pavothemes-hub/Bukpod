# Install Theme and Activate

Installing the **Bukpod** theme is quick and simple.

**Step 1 -** Log in to your **WordPress Dashboard**.

**Step 2 -** Go to **Appearance → Themes → Add New → Upload Theme**.

**Step 3 -** Choose the `bukpod.zip` file from the package you downloaded, then click **Install Now**.

**Step 4 -** Once installation finishes, click **Activate**.

{% hint style="danger" %}
After activating, WordPress will prompt you to install the required plugins — **Pavo Core**, **Elementor**, and **WooCommerce**. Install and activate all of them before moving on, otherwise Bukpod's Theme Builder, custom widgets, and shop features will not work.
{% endhint %}
