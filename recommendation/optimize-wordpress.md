# Optimize WordPress

Tired of slow website loading time?

Don't worry! We will help you boost site speed by:

* Install a Cache Plugin for WordPress. We recommend [Autoptimize](https://wordpress.org/plugins/autoptimize/) and [WP Super Cache](https://wordpress.org/plugins/wp-super-cache/).
* Install an Image Compress Plugin. We recommend WP Smush It.​
* Install a Database Optimizer Plugin We recommend WP Optimize​
* Reduce the number of posts on the page
* Reduce the number of unnecessary plugins.
* Use more icons and less images.‌

Following the "Settings" guide below of the 2 plugins [Autoptimize](https://wordpress.org/plugins/autoptimize/) and [WP Super Cache](https://wordpress.org/plugins/wp-super-cache/).

&#x20;ensures that your site will achieve Speed Optimization on Mobile, Desktop and Tablet also.
