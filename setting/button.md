# Button

This is Elementor's **Buttons** default style, under **Site Settings**. It sets the default look for every Button widget on the site (font, colors, border, padding, hover behavior).

**Step 1** - you can go to any elementor edit page.

<figure><img src="../.gitbook/assets/Screenshot 2024-07-30 at 15.04.36 (1).png" alt=""><figcaption></figcaption></figure>

**Step 2** - Site Settings >> Button >>

<figure><img src="../.gitbook/assets/Screenshot 2024-09-16 at 16.48.14.png" alt=""><figcaption></figcaption></figure>

**Step 3** - Configure the default button style:

* **Typography** - default font for button text.
* **Text Color** / **Background Color** - default (normal state) colors.
* **Hover Color** / **Hover Background Color** - colors applied on hover/focus.
* **Border Type, Width, Color, Radius** - default border.
* **Padding** - default inner spacing.
* **Hover Animation** and **Transition Duration** - the animation played and how long it takes when hovering a button.

**Step 4** - Click **Update** to save.

{% hint style="info" %}
On each individual **Button** widget, the **Content** tab has a **Type** control - `Default`, `OutLine`, `Info`, `Success`, `Warning`, `Danger`, `Link`. `Default`/`OutLine`/`Link` follow the colors you set here in Site Settings; `Info`/`Success`/`Warning`/`Danger` are fixed semantic colors (blue/green/yellow/red) meant for status-style buttons and ignore the Site Settings colors.
{% endhint %}
