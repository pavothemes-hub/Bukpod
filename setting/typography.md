# Typography

This is Elementor's **Global Fonts** (Typography), under **Site Settings**. Bukpod ships with 5 font roles pre-configured; changing one here updates every widget/place across the site that uses that role.

**Step 1** - You can go to any elementor edit page.

<figure><img src="../.gitbook/assets/Screenshot 2024-07-30 at 15.04.36 (1).png" alt=""><figcaption></figcaption></figure>

**Step 2** - Site Settings >> Typography >>

<figure><img src="../.gitbook/assets/Screenshot 2024-09-16 at 16.46.44.png" alt=""><figcaption></figcaption></figure>

**Step 3** - Edit each font role:

1. **Primary** - the main heading font (page titles, `H1`).
2. **Secondary(Heading)** - other headings (`H2`-`H6`) and section titles across widgets.
3. **Text** - body/paragraph text, descriptions.
4. **Accent** - buttons and call-to-action elements.
5. **Tertiary** - a spare font role you can assign manually on any widget's Typography control when you need something different from the 4 roles above.

For each role you can set **Font Family**, **Weight**, **Size**, **Transform**, **Style**, **Decoration**, **Line-Height**, and **Letter Spacing**.

<figure><img src="../.gitbook/assets/Screenshot 2024-09-16 at 16.46.03 (1).png" alt=""><figcaption></figcaption></figure>

**Step 4** - Click **Update** to save.

{% hint style="info" %}
Any widget's own **Style** tab can still override these globals for that one instance - click the small globe/link icon next to a Typography control and choose a different global font, or "Custom" to set it manually.
{% endhint %}
